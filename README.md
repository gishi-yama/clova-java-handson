# ハンズオンの内容

[line/clova-cek-sdk-java](https://github.com/line/clova-cek-sdk-java) を使って、Java アプリケーション（Spring Bootフレームワーク）でClovaスキルを作ってみる

**次の環境で動作確認をしています**

- macOS 10.14.6
- AdoptOpenJDK 11.0.4+11
- maven 3.6.2
- IntelliJ 2019.2.3
- ngrok 2.3.5 (mac os)
- SDK of the Clova CEK for Java 1.2.0
- Spring Boot 2.1.9
- Clova Friends Mini
