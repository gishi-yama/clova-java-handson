# ハンズオンの内容

[line/clova-cek-sdk-java](https://github.com/line/clova-cek-sdk-java) を使って、Java アプリケーション（Spring Bootフレームワーク）でClovaスキルを作ってみる

**次の環境で動作確認をしています**

- macOS 10.12.6
- Oracle JDK 1.8.0_192 & Oracle OpenJDK 11.0.1
- maven 3.6.0
- IntelliJ 2018.2.7
- ngrok 2.2.8 (mac os)
- SDK of the Clova CEK for Java 1.0.0
- Spring Boot 2.1.1
- Clova Friends Mini
